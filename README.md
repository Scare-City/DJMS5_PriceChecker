# DJMS5_PRICECHECKER

A lightweight price checker tool for the DJMS5 project, designed to verify and compare item prices quickly.

## Features

- Check current prices for listed items
- Compare price changes over time
- Simple setup and usage

## Installation

1. Clone the repository.
2. Install any required dependencies.

## Usage

1. Run the price checker script.
2. Provide the item list or target IDs as needed.
3. Review the output to see price results.

## Configuration

- Update the configuration file or script settings with your preferred item list and source endpoints.
- Ensure network access to price sources.
